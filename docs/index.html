行业情况

  • 绝大多数市场参与者亏钱
    • 在杠杆交易中只有5%的连续盈利
    • 在股票交易中只有20%的连续盈利
  • 市场参与者存在最多的三种人
    • 大师: 可能有真的方法。喜欢把简单的包装起来，故弄玄虚。
    • 骗子: 一根K线打天下，一个指标赚上百万
    • 偏执狂: 失败的多偏执，成功的更谦虚。不理解，也不想理解，还喜欢评判。
  • 为什么？
    • 金融市场本身不存在圣杯。
    • 参与金融交易门槛低: 一般来说进入门槛越低，赚钱越难。类似金融交易，赌博，彩票
      • 无时间限制
      • 无专业要求
      • 无纪律要求
      • 无资金要求和限制
    • 进入门槛高的职业: 工程师，医生，律师，高翻，科学家，专业交易员，分析师

金融市场本质

  • 市场行情
    • 因为市场是不可知不可测的，所以不存在一种能解释完全行情的方法。所有的方法都有周期性和概率性。
    • 市场不是数学公式，是反映参与者心理。不同的人，资金不同，周期不同，面对同一张行情图，多空完全不同。
    • 市场是一个混沌系统，像是天气。预测不准市场就像天气预报不可能准确。
      • 黑天鹅现象: 市场不可能全被知道，所以会经常出现没有出现过的现象。
      • 灰犀牛现象: 行情是参与者情绪的表达，如果情绪不一致就会出现横盘，如果情绪一致就会出现趋势
        • 情绪有两种: 贪婪和恐惧
  • 市场分析
    • 盘口分析，技术分析和基本面分析是现在最有效的三种分析方式。本质是三个维度去理解行情，但都是不全面的，好似盲人摸象。
分析方法 解释 择时 使用场景 对比
基本面分析 通过从宏观，行业和公司价值来判断股价是否背离。实际上一个公司的价值是交易者愿意给的价钱而不是本身的特性。 买什么 长线 主流，效果虚无
技术面分析 历史成交单形成K线。通过对历史的K线特点分析来判断未来可能情况。图表决定一切 周和天级别入口 中短线 非主流，效果好
盘口分析 通过分析挂单和吃单来了解大资金最新的动向。通过跟随大资金来获得稳定盈利。利润空间最小，盈利率最高。 分钟入口 超短线交易，剥头皮交易，高频交易，炒单 极小众，极有用
  • 赚钱和赌博
    • 买卖金融产品的正差价就是盈利
    • 参与者影响行情唯一途径是挂单或者下单。
      • 价格是最直接，最重要的研究点
    • 由于市场不可知特性，参与者必定会感受到随机激励。如果克服不了随机激励就会产生赌博式的快感和悲剧。
      • 设计自己的交易系统，坚定执行。
      • 控制风险，亏损不能超过2%
        • 严格止损 (点位止损和时间止损): 你只要不止损，就成了别人的猎物
        • 仓位管理
        • 风险对冲

成为专业交易员

  • 持续学习金融交易认知金字塔

  • 锻炼自己的交易技巧

    • 学习期: 13买点，交易区间，盘口规律
    • 练习期: 止损，止盈，大小区间
    • 大量实盘和复盘练习盘感
    • 赚钱期:
      • 少做: 控制风险
      • 多做: 设最大亏损。比如设置每天最大亏2%

  • 根据自己的性格来设计，设计交易系统

  • 云聪十三个买卖点1，价值连城

  • 日内交易计划
    • 目的: 第一稳定，第二收益率。稳定和盈利不可兼得。
    • 纪律:
      • 早盘大亏或小盈利，下午不做。方法不适合行情。
    • 步骤:
      • 选品种:
        • 手续费低
          • 美国的迷你股指
        • 流动性好
          • 期货中的主力合约，持仓量大，成交量大
        • 盘口规律明显
          • 大单进入，股价明显上涨
          • 大单下砸，股价明显下降
      • 研究波动率:
        • 历史日K线，30分钟K线，日内分时图复盘: 趋势线，形态，支撑压力2，13买点
        • 统计分析日内波动率
          • 大区间，一波上涨或下跌多少个点
          • 小区间，一波弱势和强势的反弹或回调多少个点
          • 一波盘整多少个点
        • 使用
          • 判断大小区间
            • 大区间，在波动率基础上做趋势
            • 小区间，在波动率基础上做波动。某只股票历史上的反弹是100个点，今天这只股票反弹到100个点的时候马上走。不要贪婪
        • 基本面研究做参考
      • 择时: 避开死亡时段
        • 趋势交易法:
          • 震荡时段是死亡时段 (中午)
          • 开盘后一小时和收盘前一小时容易出行情
        • 行情甄别: 好做，不好做，一般，不知道
          • 好多: 多做
          • 不好做，避免做，小心报复心理:
            • 大V反转: 直上直下的V型反转
            • 极小盘整: 波动率很小的盘整
            • 倒三角形: 很少
      • 制定交易计划
        • 定方向: 今天是要做多，做空或双向做 还是不做。一旦决定，不要轻易改变
          • 顺大逆小: 如果大势是涨，尽量做涨。如果大势是跌，尽量做跌。
          • 逆势衰竭
          • 明确反转: Pinbar, K线组合，二次测试，形态突破
        • 画盘中交易的区间:
          • 顺势突破
          • 回调: 支撑，压力，量能积累，分价图，中轴线位(箱体)，机构订单流，点位，Pinbar(针线), 形态
        • 盘前分析:
          • 高开: 做？不做？
          • 平开: 做？不做？
          • 低开: 做？不做？
      • 开盘后
        • 定大小区间，应用对应的波动率，决定持仓长短
        • 等待: 飞云十三个买点+盘口确认 (跟随大单) + 盈亏比
        • 情绪或身体不好，不交易
